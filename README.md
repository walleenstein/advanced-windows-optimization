## ⚙️ Group Policy (gpedit)

Shortcut to Windows Group Policy Editor.

This tool allows you to:

* Control internal system behaviors
* Disable unnecessary features
* Reduce background processes
* Adjust functionalities such as telemetry and updates

**Purpose:** Gain deeper control over Windows behavior.

---

## 🧠 Windows Registry (regedit)

Shortcut to the Registry Editor.

The registry stores internal system configurations.

Through it, you can:

* Adjust system performance
* Reduce delays and latency
* Improve overall responsiveness
* Change Windows component behavior

**Purpose:** Apply low-level optimizations directly to the system.

---

## ⚡ Advanced Scripts (nananrun)

Contains commands and advanced optimization scripts.

These scripts can:

* Adjust power management
* Improve system timing
* Optimize network behavior
* Apply configurations not available in the GUI

**Purpose:** Perform deeper technical adjustments in an automated way.

---

## 🚀 Usage Guide

1. Install all files from the `/installers` folder
2. Install the browser from the `/browser` folder
3. Configure mouse and keyboard
4. Configure Open-Shell
5. Apply adjustments in `gpedit`
6. Apply adjustments in `regedit`
7. Run the scripts from the `/nananrun` folder
